# Proxy vs Reverse Proxy

## Overview

Proxies and reverse proxies are intermediary servers that act as gateways between clients and servers. Understanding their differences and use cases is important for designing secure and scalable systems.

## Prerequisites

Basic understanding of client-server architecture and HTTP protocols.
