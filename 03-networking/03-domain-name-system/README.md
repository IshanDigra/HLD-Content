# Domain Name System

## Overview

The Domain Name System (DNS) is a hierarchical distributed naming system that translates human-readable domain names into IP addresses. It's essential for how users access websites and services on the internet.

## Prerequisites

Understanding of IP addresses and basic networking concepts.
